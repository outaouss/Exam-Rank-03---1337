# EXAM-RANK-03-1337-KHO

A collection of 14 Python exercises covering common algorithmic and string manipulation problems.

## Structure

```
EXAM-RANK-03-1337-KHO/
├── Subjects/       # Empty stubs — solve these yourself
└── Answers/        # Reference solutions for each exercise
```

- **Subjects** contains one file per exercise with the problem description and an empty function/class stub ready to be filled in.
- **Answers** contains the completed solutions.

## Exercises

| # | File | Topic |
|---|------|-------|
| 01 | `isAnagram.py` | Check if two strings are anagrams |
| 02 | `twoSum.py` | Find two indices that sum to a target |
| 03 | `groupAnagrams.py` | Group a list of strings by anagram |
| 04 | `topKFrequent.py` | Return the K most frequent elements |
| 05 | `isPalindrome.py` | Valid palindrome (case-insensitive, alphanumeric only) |
| 06 | `isValid.py` | Valid parentheses / bracket matching |
| 07 | `customSort.py` | Sort strings by length, alphabet, then vowel count |
| 08 | `mergeList.py` | Merge two lists into a single sorted list |
| 09 | `rotateList.py` | Rotate a list to the right by k steps |
| 10 | `reverseMatrix.py` | Reverse each row of a 2D matrix |
| 11 | `shiftString.py` | Caesar cipher — shift letters by k positions |
| 12 | `alternateCase.py` | Alternate uppercase/lowercase on letters only |
| 13 | `convertBase.py` | Convert a number string between any two bases (2–36) |
| 14 | `patternTracker.py` | Count consecutive digit pairs where second = first + 1 |

## How to Use

Work through the files in `Subjects/`, implement the function or class, and run the file to check your output against the expected results printed at the bottom of each file.

```bash
python3 Subjects/01_isAnagram.py
```

Once done, compare your solution with the corresponding file in `Answers/`.

## Requirements

- Python 3.10+
- No external libraries needed
